## 2. User and Group Management

### Create a group

```bash
sudo groupadd devteam
```

### Create users with home directory and bash shell

```bash
sudo useradd -m -s /bin/bash alice
sudo useradd -m -s /bin/bash bob
```

### Add users to the group

```bash
sudo usermod -aG devteam alice
sudo usermod -aG devteam bob
```

### Set user passwords

```bash
echo "alice:password" | sudo chpasswd
```

## 3. File and Directory Setup

### Create project directory and file

```bash
sudo mkdir -p /srv/project
echo "Top Secret Project File" | sudo tee /srv/project/data.txt
```

### Change group ownership

```bash
sudo chown :devteam /srv/project/data.txt
```

### Set file permissions

```bash
sudo chmod 600 /srv/project/data.txt
```

**Result:**

- Only the file owner can read and write
    
- Group and others have no access

## 5. Crontab Practice

### View cron jobs

```bash

```

```bash

```

### Cron job entry

```bash

```
