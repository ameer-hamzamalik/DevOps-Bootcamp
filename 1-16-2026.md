-configure github usename and email on ubuntu
```
git config --global user.name "Your Name"

git config --global user.email "your_email@example.com"

```
-github sync with obsidian with ssh key on ubuntu
```
ssh-keygen -t ed25519 -C "your_email@example.com"
```
```
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

```
```
cat ~/.ssh/id_ed25519.pub

```
- Copy the entire output (including the `ssh-ed25519` part and your email).
    - Log in to GitHub, go to **Settings** > **SSH and GPG keys** > **New SSH key**.
    - Give the key a descriptive title and paste the copied public key into the "Key" field. Click **Add SSH key**.
- **Test the connection:** In your terminal, run:
```
ssh -T git@github.com

```
